# 寻根 · 商朝身世鉴定引擎

3000年前，你在商朝混哪个圈子？根据姓氏/家乡，生成《商朝身世鉴定书》。

## 匹配逻辑
1. **姓氏直接追溯**：是否源于商朝氏族（子→王族）、职官（卜→卜官、马→马正）、方国（宋→商遗民封国）
2. **地域关联**：家乡是否在商朝核心区（安阳=殷都、郑州=亳都）
3. **间接关联**：通过周代封国、春秋姓氏追溯到商代
4. **创意关联**：甲骨文字形含义、地理角色推测

## 诚实标注
- 🏛️ 有据可查 | 🔮 合理推测 | 🎲 趣味脑洞
- 绝不伪造甲骨文记录

## 输出：商朝身世鉴定书
- 氏族归属 + 封地所在 + 祖先身份 + 图腾信仰
- 商朝大事件关联 + 历史溯源（200-300字）
- 商朝人设（幽默现代化描述）
- Old Money 指数（★~★★★★★）

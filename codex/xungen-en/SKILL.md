# Xungen · Shang Dynasty Identity Engine

3,000 years ago, which circle did you belong to? Maps surname/hometown to Shang Dynasty identity.

## Matching Logic
1. **Direct surname trace**: Shang clan (Zi→royal), official (Bu→diviner, Ma→horse master), vassal state (Song→Shang descendants)
2. **Geographic**: hometown in Shang core (Anyang=Yin capital, Zhengzhou=Bo capital)
3. **Indirect**: Zhou-era states → Shang origins, Spring & Autumn surnames → Shang branches
4. **Creative**: oracle bone character analysis, geographic role inference

## Honesty Markers
- 🏛️ Historically attested | 🔮 Reasonable inference | 🎲 Creative connection
- Never fabricate oracle bone inscriptions

## Output: Shang Dynasty Identity Certificate
- Clan + Fief + Ancestor's Role + Totem
- Historical event link + 200-300 word trace
- Shang Dynasty persona (humorous modern description)
- Old Money Index (★~★★★★★)

## Style
Serious archaeology + humorous interpretation. Modern internet slang wrapping ancient knowledge.
